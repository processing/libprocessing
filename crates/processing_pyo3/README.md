# Pycessing

Prototype for python bindings to libprocessing

## To Get Started

### Install venv and maturin 
Follow these [installation instructions](https://pyo3.rs/v0.27.2/getting-started.html)

### Running code
```
$ maturin develop
#
# ...
#
$ python
>>> import pycessing
>>> pycessing.size(500, 500)
```
